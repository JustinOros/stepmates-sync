# StepMates Sync

Mobile companion apps for syncing steps to [StepMates](https://justinoros.github.io/step-tracker).

## Android
See [android/README.md](android/README.md) for setup and build instructions.

## iPhone (iOS)
Coming soon! In the meantime, iPhone users can use the free version of [Health Auto Export](https://apps.apple.com/us/app/health-auto-export-json-csv/id1115567069) from the App Store to create an Automated Export which syncs steps automatically to StepMates.
